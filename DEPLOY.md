# Инструкция по деплою бота на бесплатный хостинг

## Вариант 1: Railway.app (РЕКОМЕНДУЕТСЯ) ⭐

Railway предоставляет бесплатный план с $5 кредитов в месяц, что достаточно для Telegram бота.

### Шаги:

1. **Зарегистрируйтесь на Railway**
   - Перейдите на https://railway.app
   - Войдите через GitHub (удобнее всего)

2. **Создайте новый проект**
   - Нажмите "New Project"
   - Выберите "Deploy from GitHub repo"
   - Выберите ваш репозиторий `telegram-roulette`
   - Railway автоматически определит, что это Python проект

3. **Настройте переменные окружения**
   - В настройках проекта найдите "Variables"
   - Добавьте переменные:
     ```
     BOT_TOKEN=ваш_токен_бота_от_BotFather
     WEB_APP_URL=https://yba888888-oss.github.io/telegram-roulette/roulette.html
     ```
   - Или ваш URL GitHub Pages, если он другой

4. **Запустите деплой**
   - Railway автоматически начнет деплой
   - Дождитесь завершения (обычно 2-3 минуты)
   - Бот будет работать постоянно!

5. **Проверьте логи**
   - В разделе "Deployments" → "View Logs" вы увидите логи бота
   - Должно быть сообщение "Бот запущен..."

### Важно:
- Railway дает $5 бесплатных кредитов в месяц
- Для Telegram бота этого более чем достаточно
- Если кредиты закончатся, бот остановится (но можно добавить карту для продолжения)

## Вариант 2: Render.com (АЛЬТЕРНАТИВА)

Render также предоставляет бесплатный план.

### Шаги:

1. **Зарегистрируйтесь на Render**
   - Перейдите на https://render.com
   - Войдите через GitHub

2. **Создайте новый Web Service**
   - Нажмите "New +" → "Web Service"
   - Подключите ваш GitHub репозиторий `telegram-roulette`

3. **Настройте сервис**
   - **Name**: `telegram-roulette-bot` (любое имя)
   - **Environment**: `Python 3`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python bot.py`
   - **Plan**: Выберите "Free"

4. **Добавьте переменные окружения**
   - В разделе "Environment" добавьте:
     ```
     BOT_TOKEN=ваш_токен_бота_от_BotFather
     WEB_APP_URL=https://yba888888-oss.github.io/telegram-roulette/roulette.html
     ```

5. **Запустите деплой**
   - Нажмите "Create Web Service"
   - Render начнет деплой
   - Дождитесь завершения

### Важно:
- На бесплатном плане Render бот может "засыпать" после 15 минут бездействия
- Но для Telegram бота это обычно не проблема, так как он постоянно получает обновления
- Если бот заснет, он проснется при следующем запросе

## Вариант 3: PythonAnywhere (ПРОСТОЙ ВАРИАНТ)

PythonAnywhere предоставляет бесплатный план с ограничениями.

### Шаги:

1. **Зарегистрируйтесь на PythonAnywhere**
   - Перейдите на https://www.pythonanywhere.com
   - Создайте бесплатный аккаунт

2. **Загрузите код**
   - В разделе "Files" загрузите файлы проекта
   - Или используйте Git: `git clone https://github.com/yba888888-oss/telegram-roulette.git`

3. **Установите зависимости**
   - Откройте "Consoles" → "Bash"
   - Выполните: `pip3.10 install --user python-telegram-bot python-dotenv`

4. **Создайте файл .env**
   - В "Files" создайте файл `.env` в корне проекта
   - Добавьте:
     ```
     BOT_TOKEN=ваш_токен_бота_от_BotFather
     WEB_APP_URL=https://yba888888-oss.github.io/telegram-roulette/roulette.html
     ```

5. **Создайте задачу (Task)**
   - Перейдите в "Tasks"
   - Создайте новую задачу
   - Команда: `python3.10 bot.py`
   - Запустите задачу

### Важно:
- На бесплатном плане PythonAnywhere задачи могут останавливаться
- Но можно настроить автоматический перезапуск

## Проверка работы бота

После деплоя:

1. Откройте Telegram и найдите вашего бота
2. Отправьте команду `/start`
3. Бот должен ответить приветственным сообщением
4. Проверьте логи на платформе, чтобы убедиться, что бот работает
